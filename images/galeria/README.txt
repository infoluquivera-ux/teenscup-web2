GALERÍA DE FOTOS
==================

Para que las fotos aparezcan en la galería de la página:

1. Renombrá tus fotos como:
   - galeria-1.jpg  ← esta es la GRANDE (foto principal, elegí la mejor)
   - galeria-2.jpg
   - galeria-3.jpg
   - galeria-4.jpg
   - galeria-5.jpg
   - galeria-6.jpg

2. Ponelas en esta carpeta (images/galeria/)
3. Listo. Las que pongas se ven, las que falten muestran un placeholder.

RECOMENDADO:
- Formato: JPG (más liviano que PNG)
- Tamaño: 1200x900 px aproximadamente (proporción 4:3)
- Peso: comprimirlas a 200-400 KB cada una
- Calidad: las mejores fotos del torneo (acción, festejo, equipos)

Para comprimir fotos pesadas, usá:
- tinypng.com (gratis, online)
- squoosh.app (gratis, de Google)
